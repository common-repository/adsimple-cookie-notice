== Description ==

AdSimple Cookie Notice hilft Ihnen Ihre Besucher einfach und elegant uber die Verwendung von Cookies zu informieren und bietet die Möglichkeit auf die Datenschutzerklarung zu verlinken.
Mit der AdSimple Cookie Notice - und mit dem AdSimple Impressum- und Datenschutz-Generator - helfen wir Ihnen sich für die EU-DSGVO, die EU-Datenschutz-Grundverordnung, zu rüsten.

== Installation ==

1.  Laden Sie das Plugin in Ihr '/wp-content/plugins' Verzeichnis.
2. Aktivieren Sie das Plugin uber das 'Plugins' Menu in WordPress.
